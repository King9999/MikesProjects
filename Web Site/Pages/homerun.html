Home Run is a Flash game created for the Game Prototype Challenge v12. Send a ball flying as far as you can by mashing buttons as fast as you can. If you're an expert masher, you should be able to max out the meter!

This game started out as a prototype for my Flash class, but based on GPC v12's theme, I thought this game was suitable.
