Years ago, I took an interest in making fighting game sprites in the same style that Capcom used for the Street Fighter Alpha games. I even used the same palette limitations, where each character had 16 colours max. I used to take requests for sprites; my only condition was that the sprite could not be from a fighting game. It was basically a "how would this character look in a 2D fighting game?" scenario. Kanji is my most recent sprite, but note that I started working on him before Persona 4 Arena was announced. I've stopped since, but I had him lying around, so I thought I'd show him off in his incomplete state.
