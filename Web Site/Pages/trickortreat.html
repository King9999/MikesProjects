Trick Or Treat is a 2-player Flash game created for the Game Prototype Challenge back in October 2011. The objective is to collect lots of candy from houses within the time limit. Players can perform "Tricks" to steal candy from other players. Each costume has different attributes.

P1 Controls: WASD to move, C to use trick. P2 Controls: directional keys to move, Right Shift to use trick.

On the costume select screen, use the left and right arrow keys to move the cursor, and the spacebar to select costume.
