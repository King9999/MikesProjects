This is a card game I made nearly 10 years ago. I still have it kicking around, so I thought I'd share it. Most of my experience with C++ is self-taught, and this game is the result of what I learned. Looking at the code now, I can see where I could improve the code (mainly my implementation of the cards). There are no graphics involved, so everything runs through a command line window, but it does allow for AI players. The card game is based on a game called 99, something that I played in elementary school. You play it using a regular deck of cards.
