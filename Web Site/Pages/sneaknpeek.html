Sneak 'n' Peek is a Unity game that my team created for the Global Game Jam in January 2012. A wealthy mystery client wants you, a famous thief, to retrieve a valuable item for him...but he wants you to guess what it is. Use the given clue to figure out which item the client wants, then escape!

Controls: WASD to move, Left mouse button to pick up item, Space to jump, Shift or Caps Lock to run really fast.
