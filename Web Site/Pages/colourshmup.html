I created this game for Game Prototype Challenge v16. It's been a while since I made a game for the event, and I had time, so why not? The theme this time was "four colours" and "absorption." I brainstormed and came up with a shmup (shoot 'em up). I thought the game came out well despite some bugs, and I think the game fits the theme well without it feeling forced. CONTROLS: WASD to move ship/move menu cursor, Space to shoot/select menu, directional keys to change colour.
